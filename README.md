# 🚀 DPlayer

Музыкальный плеер, позволяющий быстро структурировать и слушать все .mp3 файлы на компьютере.

## 🛠️ Как запустить программу

Для запуска проекта не нужно ничего устанавливать или компилировать. Просто следуйте инструкции:

1. Скачайте архив проекта из вкладки**[Releases](https://github.com/Delinor-Hens/DPlayer/releases)**.
2. Распакуйте скачанный ZIP-архив в любую папку.
3. Найдите файл `DPlayer.exe` в распакованной папке.
4. Дважды кликните по нему для запуска.

## 📋 Системные требования

* **Операционная система:** Windows 10 / 11 (64-bit)
* **Дополнительно:** Нет требований
